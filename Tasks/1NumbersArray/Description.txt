ТАСК. ЧИСЛОВЫЕ МАССИВЫ

Задача такая есть поле для ввода исходных данных (числа разделенные запятой), кнопка Вычислить и поля для вывода результата
Задания

(Easy Level)
1) Посчитать сумму четных чисел и вывести в поле Сумма четных чисел.
2) Посчитать НОД между наибольшим и наименьшим числом и вывести в поле "НОД максимума и минимума"
3) Вывести все числа которые кратны первому числу в поле "Кратные числа"

(Normal Level)
Добавлены правила (также касаются заданий из Easy level):
	- Поле ввода должно валидироваться при нажатии на кнопку вычислить (если там не числа, показывать сообщение о некорректном вводе и не делать никаких вычислений
	- Обработка ситуации в задании если происходит деление на ноль (выводить в поле результата задания сообщение об ошибке делении на ноль)
	- Все поля результатов должны быть readonly.
	- Для каждого задания должна быть как минимум одна js функция
	- Должна быть отдельная функция получения данных из поля ввода исходных данных
	- Должна быть отдельная функция которая проверяет на валидность исходные данные.
1) Извлечь простые числа и вывести в поле "Простые числа", числа извлекаются следующим способом: простое число отбирается в диапазоне между двумя соседними числами в поле ввода 
(если чисел в диапазоне оказалось много, то берется наибольшее, если соседние числа равны или простых чисел не найдено, то отбирается единица). 
2) (В задании нельзя использовать индексацию типа arr[0], arr[6] и т.п. поищи js функции другие).
Из исходных чисел сделать копию данных в другой массив (назову пока его m1), из данных m1 сделать массив m2 элементы которого будут следующими:
	- первый элемент: сумма первых двух элементов m1 (эти элементы должны исчезнуть из m1)
	- второй элемент: сумма длины и минимального элемента m1
	- третий элемент: центральный элемент m1 (элемент должен исчезнуть из m1)
	- четвертый элемент: сумма первого и последнего элемента m1 (элементы тоже должны исчезнуть из m1)
	- пятый элемент: индекс максимального числа m1
	- шестой элемент: длина m1
Вычисления не должны производиться если исходных данных не хватает для задания (выводить сообщение о том что данных не хватает для задания)
Вывести значения m2 в поле "Массив преобразование исходных данных".
